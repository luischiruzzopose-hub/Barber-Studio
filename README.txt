Para subir a Vercel:
1. Subí la carpeta a GitHub.
2. Importá el repositorio en Vercel.
3. Deploy.

Si querés cambiar el link de TidyCal:
En index.html buscá:
data-path="luischiruzzopose/reserva"
y cambiá ese valor por el nuevo usuario/ruta de TidyCal.
